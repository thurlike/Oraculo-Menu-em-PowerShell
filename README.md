# Oraculo-Menu-em-PowerShell
Oráculo Menu em PowerShell é um menu interativo criado para centralizar e padronizar tarefas administrativas em ambientes Windows. Ele acelera rotinas de suporte e infraestrutura, reduz erros operacionais e facilita diagnósticos, auditorias e manutenções, promovendo automação responsável e eficiência no dia a dia de TI.
